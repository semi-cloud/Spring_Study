## :pencil2: 단축키
 > Intellij 단축키 모음

 + `ctrl + E` : 최근 파일 사용목록 조회
 + `ctrl + shift + space` : 코드 자동완성
 + `ctrl + P` : 현재 메서드의 파라미터 조회
 + `ctrl + alt + V` : 변수 추출
 + `Alt + Insert` : 생성자, Getter/Setter 등 코드 자동생성(Lombok으로 대체 가능)
 + `shift + ctrl + enter` : 다음 줄로 이동
 + `ctrl + o` : 메소드 오버라이딩
 + `ctrl + alt + B` : 해당 코드 구현부로 이동
 + `ctrl + D` : 라인 복제
