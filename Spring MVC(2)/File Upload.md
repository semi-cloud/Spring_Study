# File Upload

## 파일 업로드 개요
 + 🤨 `HTML FORM`에 문자와 바이너리 데이터(파일)를 동시에 전송해야 하는 상황 발생
 + `multipart/form-data` : HTTP의 전송 방식으로, 다른 종류의 여러 파일과 폼의 내용을 함께 전송 가능 
   + Form 태그에 `enctype="multipart/form-data"` 지정하고 사용
   + `Content-Disposition` 항목별 헤더를 통해 항목 구분
